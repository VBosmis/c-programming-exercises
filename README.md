# C Programming Exercises

This repository contains three small C console programs created for academic practice.  
Each file demonstrates a different topic in programming, logic, and user interaction.

## Included Programs

### • EasterCalc.c
Calculates the date of Orthodox Easter for any year between 1900–2099, including input validation.
Source based on: EasterCalc.c

### • UserRights.c
Simulates a simple permission system using bitwise operations (read, write, execute), with interactive menu control.
Source based on: UserRights.c

### • DeckCardGen.cpp
Generates and displays a random playing card using ASCII formatting and color changes.
Source based on: ΕργασιαΕξαμηνου.cpp

## Notes
All programs were created and tested in a Windows environment and use Windows-specific commands.
